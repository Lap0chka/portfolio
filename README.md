Django Portfolio Website
Welcome to the repository for my personal portfolio website, built using the Django framework.
This project showcases my skills and projects, featuring functionalities such as logging,
data models, blog and localization in four languages: English, German, Russian, and Ukrainian
and enforced code quality standards using tools like Black, Flake8, Mypy, Isort, and Pre-commit, 
comprehensive test coverage for each application, and APIs for retrieving portfolio items and blog posts and more.

Project Overview

This portfolio website includes the following sections:

Home: A brief introduction and navigation links.
About: Detailed information about my background and skills.
Projects: A gallery of my completed projects with descriptions and links.
Blog: Articles and posts related to my work and interests.
Contact: A contact form for visitors to reach out to me.
Technologies Used

Django: Backend web framework.
SQLite: Default database for development.
Bootstrap: Frontend framework for responsive design.
Pillow: Image processing library for handling project images.
Black: Code formatter that enforces consistent style.
Flake8: Tool for checking the style and quality of Python code.
Mypy: Static type checker for Python.
Isort: Utility to sort imports in Python files.
Django REST Framework: Toolkit for building Web APIs.
Pre-commit: Framework for managing and maintaining multi-language pre-commit hooks.
Features

Dynamic Content Management: Manage site content through Django's admin interface.
Blog Functionality: Create, edit, and display blog posts with categories and comments.
Localization: Support for English, German, Russian, and Ukrainian languages.
Code Quality Enforcement: Automated code formatting, linting, type checking, and import sorting.
Responsive Design: Ensures optimal viewing experience across devices.
Contact Form: Allows visitors to send messages directly through the site.
RESTful APIs: Retrieve portfolio items and blog posts via GET requests.
